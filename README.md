# Aryan Mishra — Portfolio

A clean, responsive, and animated portfolio website for Aryan Mishra, a Data Scientist and Web Developer. It showcases education, skills, and selected projects including an AI website generation tool, sample sites, utilities, and multiple Flask/FastAPI projects.

## Overview
- Modern, formal design with subtle glass and gradient accents
- Fast, responsive layout for mobile, tablet, and desktop
- Animated buttons and cards with scroll‑reveal effects
- Sticky, accessible navigation with active section highlighting
- Sections: Hero, About, Projects, Skills, Contact
- Links to GitHub and LinkedIn
- Easy to customize hero photo and content

## Setup
1. Download or clone the repository.
2. Open index.html in any modern web browser.
3. No build steps or servers required.

Performance notes:
- Uses a single Google Font and optimized Unsplash image parameters for quick load.
- Animations are lightweight; prefers-reduced-motion is respected.

## Usage
- Replace the hero portrait image:
  - In index.html, find the .portrait background URL and replace it with your photo link.
- Update or extend projects:
  - In the Projects section, duplicate a card block to add new entries. Update the title, description, tags, and link.
- Customize colors:
  - Edit CSS variables in :root for theme colors and radii.
- Navigation:
  - Sticky navbar with smooth scrolling and active link highlight as you move through sections.
