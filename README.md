# Aryan Mishra — Portfolio

Overview
- A clean, responsive, and animated portfolio website for Aryan Mishra, a data scientist and web developer.
- Highlights education (BS in Data Science and Programming at IIT Madras; BSc (Hons) in Physics), selected projects, skills, and contact links.

Setup
1. Download or clone the repository.
2. Open index.html in any modern browser.
3. No build steps or dependencies required.

Usage
- Replace the hero image:
  - In index.html, find the element with class "hero-img" and replace its src with your photo URL.
  - Recommended width ~1400px for crisp display.
- Update email:
  - Replace contact@placeholder.example in index.html with your email address for the Email buttons.
- Customize content:
  - Edit text in the About, Skills, Projects, and Education sections directly in index.html.
  - Add or remove project cards by duplicating an existing project block in the Projects section.
- Deployment:
  - You can host this directly on GitHub Pages by pushing the repo and enabling Pages in repository settings (branch: main, folder: /root).
