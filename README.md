# Aryan Mishra — Portfolio

## Overview
A lightweight, responsive portfolio web page for Aryan Mishra. It highlights a data-focused skill set and provides quick access to contact information, including Instagram.

Key features:
- Direct Instagram link: https://www.instagram.com/aryanmishra1411/
- Skills prominently include: Excel, SQL, PostgreSQL, SQLite3, SQLAlchemy, and Data Analysis
- Accessible, mobile-friendly, and fast-loading (no external dependencies)

This repository is suitable for quick deployment on any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Setup
- No build steps are required.
- Files:
  - index.html — the complete web app (HTML/CSS/JS inline).
  - README.md — project information and license.

Local preview:
1. Download or clone the repository.
2. Open index.html in your web browser.

Optional (serve with a local HTTP server):
- Python 3: python -m http.server 8080
- Node.js: npx serve

## Usage
- Customize text and contact email in index.html.
- Add or modify projects in the “Highlighted Projects” section.
- Skills already include Excel, SQL, PostgreSQL, SQLite3, SQLAlchemy, and Data Analysis as requested.
- The Instagram link in the Contact section points to: https://www.instagram.com/aryanmishra1411/

Deployment:
- Upload index.html (and this README) to your static host.
- For GitHub Pages, place files on the main branch (or docs/) and enable Pages in repository settings.
